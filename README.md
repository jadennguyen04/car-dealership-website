# DriveLine Motors - Car Dealership Website

A responsive car dealership website focused on front-end development, built with HTML, CSS, Next.js, React, and TypeScript. This project emphasizes design, user experience and UI elements. Developing this web application to be ideal for showcasing premium vehicles and dealership branding.

> **Warning:** All vehicle and dealership information is stored locally in the codebase (not in a SQL database or API). This was done to simplify the project and focus on front-end design. No backend, database, or external API is used.

## Demo
- Live demo: [https://car-dealership-website-mu.vercel.app/](https://car-dealership-website-mu.vercel.app/)

## Features
* Homepage with hero section and featured vehicles
* Inventory page with detailed listings
* Vehicle detail pages with images and specs
* Contact page with embedded Google Maps
* About section with dealership info
* Responsive design for all devices
* Modern UI/UX and clean design elements

## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- pnpm (or npm/yarn)

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/jadennguyen04/car-dealership-website.git
   cd car-dealership-website
   ```
2. Install dependencies:
   ```sh
   pnpm install
   # or
   npm install
   ```
3. Run the development server:
   ```sh
   pnpm dev
   # or
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `app/` - Next.js app directory (pages, layouts, components)
- `components/` - Reusable UI components
- `public/` - Static assets (images, favicon, etc.)
- `styles/` - Global and modular CSS
- `hooks/` - Custom React hooks
- `lib/` - Utility functions

## Customization
* Update vehicle data and images in the `public/` folder and relevant pages.
* Change dealership info in the About and Contact sections.
* Modify design elements and UI components to fit your branding.
